# 20210401 作業：成績單 (transcript.cpp)

撰寫一個程式，能找出成績列表裡的：
1. 最高分的人
2. 最低分的人，
3. 平均值

直到輸入 0 為止。

必須使用以下功能：
1. `vector`
2. 包含 `auto` 與冒號的 `for`
